# ⚡ Minsar Chrome Extensions

A collection of Chrome extensions by [Minsar AI](https://github.com/minsarai).

## Extensions

| Extension | Description | Status |
|-----------|-------------|--------|
| [Minsar — Magento Admin Toolkit](./minsar-magento-toolkit/) | 10-feature command center for Magento 2 admin. Dark mode, Ctrl+K search, cache flush, compact mode. | ✅ v1.0.0 |

## Install (Developer Mode)

1. Clone: `git clone https://github.com/minsarai/chrome-extensions.git`
2. Open `chrome://extensions/` → Enable Developer mode
3. Click "Load unpacked" → select the extension folder (e.g. `minsar-magento-toolkit/`)
