# Privacy Policy — Minsar (Magento Admin Toolkit)

**Last updated:** April 20, 2026

## Data Collection

Minsar does **not** collect, store, or transmit any personal data. All data stays in your browser.

## What Minsar Stores Locally

The following data is stored in your browser using Chrome's local storage API:

- **Bookmarks:** Admin page URLs you manually save
- **History:** Last 30 admin pages you visited (URL and page title only)
- **Settings:** Your environment labels (dev/staging/production URL mappings), dark mode preference, compact mode preference

This data never leaves your browser. It is not sent to any server.

## Permissions

- **activeTab:** To detect if the current page is a Magento admin panel and inject features
- **storage:** To save your bookmarks, history, and settings locally
- **scripting:** To inject dark mode, compact mode, environment bar, and cache flush functionality
- **host_permissions (<all_urls>):** Required to work on any Magento admin URL (since admin URLs vary per installation)

## Third-Party Services

Minsar does not use any third-party analytics, tracking, or advertising services.
